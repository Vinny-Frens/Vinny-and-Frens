## Development
### Setup & Install
```shell
git clone https://github.com/Vinny-Frens/Vinny-and-Frens-Contract.git
cd Vinny-and-Frens-Contract
npm install
cp sample.hardhat.config.js hardhat.config.js
```

### Test
```shell
npx hardhat test
```

### Deploy
```sh
npx hardhat run scripts/deploy.js --network <network>
```
